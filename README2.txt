 CondoFácil — Sprint 2

Proyecto académico desarrollado con metodología Scrum.

 Descripción
CondoFácil es un sistema web para la administración de condominios.  
El Sprint 2 se enfocó en el desarrollo del módulo de cobranza y consulta de estados de cuenta, ampliando la funcionalidad base del sistema.



 Funcionalidades implementadas en Sprint 2

 Registro de usuario con persistencia básica (localStorage)
 Login dinámico
 Panel de perfil con navegación
 Registro de pagos (Módulo de cobranza)
 Visualización de pagos registrados
 Página de historial de pagos
 Manejo básico de roles



 Historias de Usuario trabajadas

 HU05: Registrar pagos de residentes (Completado)
 HU06: Historial de pagos (Completado)
 HU07: Estado de cuenta del residente (85%)

Total Sprint 2: 21 Story Points  
Completados: 18 SP



  Métricas del Sprint

 Velocidad estable respecto al Sprint 1
 85% del objetivo completado
 Burndown con leve desviación final
 3 bugs menores detectados y corregidos



  Tecnologías utilizadas

 Next.js
 React
 JavaScript
 localStorage (simulación de persistencia)
 Git y GitHub
